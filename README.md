CV estático en HTML y CSS
Autor: Dario Candia Jimenez (Estudiante de Informática)

 Este CV es una página estática con la siguiente estructura:
- Encabezado: foto a la izquierda, nombre y título a la derecha
- Sección en dos columnas:
  - Izquierda: Tecnologías, Cursos, Certificaciones
  - Derecha: Antecedentes Académicos, Antecedentes Laborales
- Pie de página con información de contacto

Archivos incluidos:
- index.html
- css/estilos.css
- assets/profile.jpg (reemplázalo por tu foto real) 

Cómo ver localmente:
- Abre index.html en un navegador (doble clic o arrastrando a la ventana del navegador)
- Opcional: sirve un servidor local en la carpeta del proyecto y abre http://localhost:8000

Cómo actualizar la foto:
- Sustituye assets/profile.jpg por la foto real. Luego haz commit y push:
  git add assets/profile.jpg
  git commit -m "CV: añade foto real en assets/profile.jpg"
  git push origin master
